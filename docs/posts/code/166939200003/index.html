算法#

一般的公司很少有算法题，出现比较高的是一些外企、一线互联网公司，像我最近去面的微软、iHerb、lexisnexis，直接在白板、白纸上写，如果写不出，可能下面的面试没法继续聊了， Leetcode上面的简单和中等的题要通过。每天抽时间多刷一刷。这样长期积累对自己的算法功底、代码的健壮性能提高很多。
